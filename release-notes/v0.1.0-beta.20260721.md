## Highlights

- The web header now reflects the authenticated user or administrator session,
  places language selection before authentication actions, and offers logout
  instead of misleading login and registration links after sign-in.
- Large user and device administration forms moved from expanding table rows to
  focused modal dialogs, keeping tables readable and actions predictable.
- Dashboard and administration layouts now use available space more
  consistently and behave better at narrower viewport widths.
- Server build, configuration, email handling, and synchronization contract
  checks were hardened for beta deployment.

## Главное

- Шапка web-интерфейса теперь отражает активную сессию пользователя или
  администратора, располагает выбор языка перед действиями авторизации и после
  входа предлагает выход вместо вводящих в заблуждение ссылок входа и
  регистрации.
- Большие формы управления пользователями и устройствами перенесены из
  раскрывающихся строк таблиц в отдельные модальные окна, поэтому таблицы
  остаются читаемыми, а действия — предсказуемыми.
- Dashboard и административные страницы последовательнее используют доступное
  пространство и лучше адаптируются к узким окнам.
- Сборка сервера, конфигурация, обработка электронной почты и проверки контракта
  синхронизации усилены для бета-развёртывания.

## Packages / Пакеты

This prerelease provides a self-contained Linux amd64 sync-server archive with
the systemd service file and installation script.

В этот prerelease входит самостоятельный архив sync-server для Linux amd64 с
файлом службы systemd и установочным скриптом.
